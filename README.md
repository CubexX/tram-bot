Where is the tram?
====
![image](https://i.imgur.com/TRM4DSe.png)

[@tram_bot](http://telegram.me/tram_bot) can show tram schedule

Installation
-------
```bash
pip3 install -r requirements.txt
```
Usage
-------
Edit main.py:
```py
TOKEN = "YOUR_TOKEN"
```
And start bot:
```bash
python main.py
```
